# Internship Radar

An internship discovery and alert platform for students targeting elite tech internships.

## V1 Goal

Build a searchable internship dashboard with:

- A seeded company and internship database
- Search and filters for SWE, AI/ML, systems, hardware, PM, research, underclassmen, and remote roles
- Company watchlist buttons
- Company-specific LinkedIn alumni search guidance
- A foundation for Supabase auth, automated daily updates, AI classification, and email alerts

## Run Locally

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Build Roadmap

1. Static dashboard with seeded roles
2. Supabase database and auth
3. Watchlist persistence
4. Admin page for manual posting review
5. Daily updater for company/job sources
6. AI classifier for role type, eligibility, urgency, and summaries
7. Email alerts through Resend
